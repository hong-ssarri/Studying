## 제어문이란
### 1) 제어문의 정의
: 프로그램의 흐름을 제어하는 문법

`프로그램의 흐름이란?`
프로그램은 여러 가지 명령으로 구성 되어있는데
가장 위에 있는 명령부터 시작해서 아래로 각각의 명령이 실행됨
**명령이 실행되는 순서**를 프로그램의 흐름이라고 할 수 있음

### 2) 제어문의 종류
(1) 조건문: 정해진 조건에 따라 흐름을 제어하는 문법
(2) 반복문: 정해진 조건에 따라 흐름을 반복하는 문법

_ _ _

## **조건문**이란
: 조건에 따라 각각 다르게 실행하도록 만들어놓은 **제어문**

### 3) 조건문의 종류
(1) If문
(2) switch문

___
___

## switch문
```java
/* 구조 */
switch(변수){
	case값1:
    	변수 == 값1일때 실행할 명령;
        break;
    case값2:
    	변수 == 값2일때 실행할 명령;
        break;
	...
    default:
    	모든 case가 아닐 때 실행할 명령;
}
```

break; 를 하지 않으면
모든 case의 명령이 실행된다
